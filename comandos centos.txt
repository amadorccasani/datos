-----------------------

yum install nombrepaquete  => installar 

listado de repositorios instalados:
----------------------------------
yum repolist

-----------

-------------

ls -la

----------------------------------
chown usuario:usuario carpeta

chown -R usuario:usuario carpeta => subcarpetas pertene
